| Prefix      | When to Use                                         | Example                                     |
| ----------- | --------------------------------------------------- | ------------------------------------------- |
| `feat:`     | New functionality or feature                        | `feat: add Azure Blob Storage upload`       |
| `fix:`      | Bug fix or incorrect behavior                       | `fix: handle missing VM tags`               |
| `refactor:` | Internal code improvement without changing behavior | `refactor: extract metadata engine`         |
| `docs:`     | Documentation only                                  | `docs: update README with architecture`     |
| `test:`     | Add or modify tests                                 | `test: add pricing connector unit tests`    |
| `build:`    | Build system, Docker, dependencies                  | `build: add Dockerfile`                     |
| `ci:`       | CI/CD pipeline changes                              | `ci: add GitHub Actions workflow`           |
| `perf:`     | Performance improvements                            | `perf: cache Azure pricing requests`        |
| `style:`    | Formatting only (no logic changes)                  | `style: reformat recommendation engine`     |
| `chore:`    | Miscellaneous maintenance                           | `chore: upgrade Python dependencies`        |
| `revert:`   | Revert an earlier commit                            | `revert: remove experimental pricing logic` |
